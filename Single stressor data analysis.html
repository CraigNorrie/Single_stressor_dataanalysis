Single stressor data analysis

Analysis of growth, oxygen consumption, and survival data collected from laboratory experiments that held diploid and triploid C. gigas under 5 levels of temperature, pH, and, disolved oxygen.

Individuals were held for a period of 4 weeks under these conditions.

Statistical analysis of the data

A linear modeling approach was used to examine differences in the mean size of each ploidy group held under different temperature treatments.

Mass

We first examined the distribution of the data to determine if it met the assumptions of normality. As the data did not fit the assumptions of normality (Figure 3) it was log transformed which improved normality (Figure 4).

As all individuals had been growing in a common environment prior to the start of the experiment we compared the size of individuals at the start of the experiment using a linear model.
This model indicated that there was a significant difference in the mass of induced triploids in the 22.5 and 7.5 °C treatments. Therefore we did not simply compare the size of individuals at the end of the experiment for each ploidy across temperature treatments.
Differences in mean size of each ploidy existed at the start of the experiment. Diploids were the smallest and mated triploids were largest (Figure 4).This is what explains the significant treatment:ploidy interaction.

We focused on changes in mass rather than shell height because of the potential mis-measureing and chipping that resulted in some shrinkage in the shell height over the course of the experiment. Using a linear mixed model that included tank as a random factor we compared the mean size of individuals from each ploidy group individually for each of the treatments at the start and the end of the experiment.

In diploids, the linear mixed model did not indicate that there was a significant time treatment interaction, or main effect of treatment. We did, however, observe a significant main effect of time (Table 2). This main effect indicated that diploids were significantly heavier at the end of the end of the experiment than at the beiging which indicated growth in all tanks (Figure 2).

In order to account for variation in mortality between tanks we performed a frailty analysis. This frailty analysis was akin to using a linear mixed model to analyse the data.

Comparisons of two methods to include fixed effects.

The frailtyEM pacakage (REFERNCE) allows you to add a frailty (i.e. random) effect to the survival analyses. The coxme Package Therneau, 2024 also allows you to add a random effect to the data, this package however allows you to use the same coxph model that you would use for pooled analysis. Pairwise comparisons for the coxme object can be done using the emmeans package Lenth et al 2024. The emmeans package allows for corrections for pairwise comparisons.
##### Coxme package

The coxme package is probably easier to use. Comparisons of the coxph model that included the random effects model indicate that it is significantly more parisomoniuos than the model that includes only fixed effects(Table 28). The pairwise comparisons within ploidy, but across treatments treatments (Table 30) indicated that diploids (blue) exhibited higher mortality at 1000 than 500 µatm above ambient. No other significant pairwise comaprisons existed.
Pairwise comparisons across ploidies, within treatments (Table 31) showed that at 500 and 750 µatm above ambient mated triploids (green) had higher mortality than both induced triploids (red) and diploids (blue). Suggesting that mated triploids were more sensitive at lower pH’s

)

# List all objects in the environment
all_objects <- ls()

# Iterate over each object
for (obj_name in all_objects) {
  # Check if the object follows the pattern frail_[colour]_[number]
  if (grepl("^frail_[A-Za-z]+_[0-9]+$", obj_name)) {
    # Extract coefficients and p-values
    coeff_p_val <- summary(get(obj_name))$coef[, c("coef", "p")]
    
    # Extract object details
    obj_details <- unlist(strsplit(obj_name, "_"))
    
    # Add object name to the results dataframe
    obj_df <- data.frame(
      Object_Name = rep(obj_name, nrow(coeff_p_val)),
      Coefficient_Name = rownames(coeff_p_val),
      Coefficient_Value = coeff_p_val[, 1],
      P_Value = coeff_p_val[, 2]
    )
    
    # Append to results dataframe
    results_df <- rbind(results_df, obj_df)
  }
