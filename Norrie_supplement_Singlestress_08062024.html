Supplementary materials: Single stressor data analysis for performance of diploids and triploids

This document presents the analysis of growth, oxygen consumption, and survival data collected from laboratory experiments that held diploid, chemically induced triploid, and mated triploid C. gigas under 5 levels of temperature, pH, and, disolved oxygen. These 3 experiments were performed sequentially.

Growth analyses

High mortality occurred in the pH experiment, thus we did not assess differences in individual growth rates due to the potential for size selective mortality to impact results. We only examine if a difference in mean mass and shell height of each pliody group in each between the start and the end of each experiment.

Differences in size at the start of each experiment

As all individuals had been growing in a common environment prior to the start of the experiment we first compared the size of individuals at the start of the experiment using a linear model that included plolidy and treatment. We ran one model for each of the experiments. This allowed us to examine the effect of ploidy on growth in a common garden environment and determine the most appropriate statistical approach.

At the start of the temperature experiment there were significant significant ploidy and treatment main effects (Table 1). This indicates that individual size differedc between treatments at the start of the experiment.

Show the code 
massgrowthlm_temp_experiment_start <- sizedat_temp %>% filter(Time == "1_start") %>% 
  lm(log(Mass)~Ploidy*Treatment, data=.)
mass_start_temp.aov <- aov(massgrowthlm_temp_experiment_start)
Show the code 
kable(anova(massgrowthlm_temp_experiment_start), digits = 3)

Post hoc comparisons revealed significant differences existed between all ploidy groups (Table 2), and between individuals in the 7.5 °C treatment which were smaller than those from the 12.5 and 22.5 C treatment.
